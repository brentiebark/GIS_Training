Basic GIS using QGIS

This course introduces participants to Free and Open Source Software for Geospatial Applications (FOSS4G). We will explore Quantum GIS as an easy to use FOSS4G tool. The materials focus on the basic interface and data exploration of QGIS.

Quantum GIS (QGIS) is a user friendly Open Source Geographic Information System (GIS) that runs on several operating systems (Linux, Unix, Mac OSX, and Windows). QGIS supports vector, raster, and database formats. QGIS is licensed under the GNU Public License.

What you will learn

At the end of the training/workshop, participants should be able to:

  • Navigate, know and use the basic interface and tools of QGIS

  • Perform basic activities such as editing and managing attribute data

  • Transform paper-based maps into GIS formats

  • Apply GIS knowledge and skills in interpreting existing geospatial datasets (for Forest Land Use Planning Purposes)
